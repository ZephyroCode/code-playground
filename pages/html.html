HTML Stuff

Never forget the documentation: MDN HTML Docs

Headings

There are 6 total tags that are used for heading elements: h1, h2, h3, h4, h5 and h6.

Every heading element is a Block Element.

Example:

        
          <h1>Hi, I'm H1</h1>

          <h2>Hi, I'm H2</h2>

          <h3>Hi, I'm H3</h3>

          <h4>Hi, I'm H4</h4>

          <h5>Hi, I'm H5</h5>

          <h6>Hi, I'm H6</h6>

Paragraphs

Paragraphs are the standard text element.

Paragraphs are Block Elements.

Example:

        
          <p>This is a paragraph!</p>

          <p>This is a second paragraph!</p>
          
          <p>This is a third paragraph!</p>

Line Breaks

Every time you use this element, anything following it will start from the next line.

Example:

        
          <p>This<br />
          is<br />
          a<br />
          paragraph!</p>

Horizontal Lines

This element draws an horizontal line from the left margin to the right margin and breaks the line.

This element is used to separate sections in a document.

Example:

        
          <p>This is a paragraph!</p>
          <hr />
          
          <p>This is a second paragraph!</p>
          <hr />
          
          <p>This is a third paragraph!</p>
          <hr />

Formatted Text

When you want some text to be show in the same format of how it's written in the HTML document, you can use the pre tag.

Pre tag is a Block Element.

Example:

        
          <pre>
            function greet(name) {
              alert(`Hi, ${name}!`);
            }
          </pre>
